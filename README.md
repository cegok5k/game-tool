# game-tool

A browser-based game tool. Purpose TBD — replace this section when the tool's scope is decided.

## Stack

TypeScript + React 19 + Vite 6, Vitest + Testing Library for tests, ESLint 9 (flat config) + typescript-eslint for linting.

## Getting started

```sh
npm install
npm run dev          # start dev server
npm run build        # typecheck + production build
npm run test         # vitest in watch mode
npm run test:run     # vitest single run
npm run lint         # eslint
npm run typecheck    # tsc -b --noEmit (project references)
```
